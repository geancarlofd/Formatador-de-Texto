Formatador de Texto

Este site tem o intuito de facilitar a formatação de seus textos, modificando letras para minúsculas ou maiúsculas.




Formate seus textos como preferir. Utilize os botões da barra superior para modificá-lo. Depois aperte o botão "Copiar", dessa forma o conteúdo do box de texto será copiado para o seu clipboard. Basta dar CTRL+V onde desejar.



Caso queira limpar o box de texto, utilize o botão "Limpar". Lembrando que seu progresso será perdido e não poderá ser restaurado.



É possível alterar o tema do site para a cor que mais lhe agradar. Clique em qualquer um dos 3 temas na barra inferior e divirta-se



Novas funcionalidades ainda serão adicionadas, para que o Formatador fique cada vez mais completo e útil.
